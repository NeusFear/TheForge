# Core Functions needed:
###############################

getRecentPosts(int ammt, int skip) :: array of post indexes:
	- returns the specified number of recent posts, and skips spefified, default get 10 recent skip 0 (this feature is to load new and not all at once)
getOnlineUsers(int ammt, string group)
	- returns ammount of specified groupes users that are online
getPostData(int index)
	- returns a json object of the data of the post, this includes
		+ poster
		+ date Posted
		+ Title of post
		+ Description
		+ tags
		+ Category forum path, Ex: (Games > minecraft > our servers)
		+ Body of post - in html format
		+ array of file paths included in post
getPostResponses(int ammt)
	- returns array of jsondata for comments on the post, in time order including:
		+ name of responder
		+ time of response
		+ body of response (unformatted, responses dont need html format, or attatchments)
getPostLikes()
	- returns array arr[0]=likes arr[1]=dislikes
getUserData(int: userid)
	- returns basic data about the user:
		+ username
		+ Name
		+ Status
		+ Path to Profile Picture
		+ xp
		+ Array of tag types
			* arr[0] - Staff roles
			* arr[1] - Donator Roles
			* arr[2] - Given roles (helper, sexy, etc.)
getProfileData(int: userid)
	- returns all data about the specified user
		+ username
		+ Name
		+ location
		+ Discord ID
		+ Steam ID
		+ Origin ID
		+ Minecraft Username
		+ Status
		+ Date Joined
		+ Path to Profile Picture
		+ Path to Banner
		+ HTML format of profile home
		+ xp
		+ Array of tag types
			* arr[0] - Staff roles
			* arr[1] - Donator Roles
			* arr[2] - Given roles (helper, sexy, etc.)
